    Unblocking Help

    General Help

    The 'Cloaked Search' option under the search box can be used to search Startpage. Startpage has these functions:

    1. Search Startpage
      • You can click the Cloaked Search option to have your entered query searched on Startpage (not Google).
      • Unfortunatelly, this will not be totally anonymous, as you will be sent to https://www.startpage.com/do/dsearch?query=[YOUR ENTERED QUERY], (so people can see what you have searched).
      • If you want to do a completely anonymous search, directly search from Startpage here.
    2. Startpage Unblocking
      • If you want to use an alternative unblocking method (if the one listed below doesn't work) using Startpage, use one of the following methods above to search Startpage. After startpage has loaded, click Anonymous View next to your intended result to unblock the website and view it completely anonymously.

    The 'Direct URL' option under the search box, or pressing 'Enter', can be used to unblock any URL entered into the search box*.

    • Basic use
      • To unblock a website, enter the full url (don't search it) into the box, then click Direct URL (main). You can also click Direct URL (alt) or Other Options:
        • Google Translate: Unblocks websites using Google translate. This is an old and known method that will probably be blocked.
        • External Website: This will copy your entered url and you can use one of the methods on the other website by simply pasting.
      • The unblocker has 2 modes:
        1. Regular mode. Just follow the steps above.
        2. Cloaked mode. Click the Turn On Cloaking link at the bottom and follow the steps above.
          • What is cloaking? Cloaking will remove the unblocked website from your history and will keep the title at 'Google' with the Google logo icon. Cloaking will not work with Startpage, so I recommend directly searching with Startpage for complete anonymity from here.
    • Known Issues
      • *If your internet filter uses wildcard blocking, then the Startpage unblocker might work, but the Direct URL unblocker will not work in any of the modes listed below.

    We also have 2 more unblocking options. If none of the options above work, you should use Remote Onworks or Remote Firefox. These options a reported to not work at many schools now.

    • Onworks is a remote virtual machine provider, and Remote Firefox is similar.
    • Onworks will allow you to connect to a virtual computer running Ubuntu in a foreign country (usually) and can do many things:
      • Browse the internet through Firefox.
      • Install and run some desktop programs.
    • Unfortunately, since you are connecting to a remote computer, your experience will be very laggy.
    • Remote Firefox will allow you to connect to a virtual computer running a slimmed down version of Linux running only Firefox in a West Virgina virtual machine (usually) and can do one thing:
      • Browse the internet through Firefox
    • Unfortunately, since you are connecting to a remote computer, your experience will be somewhat laggy, but not as laggy as Onworks.
